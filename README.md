# issue note

## `rails` 指令沒有反應

```bash
# no response
$ rails g -h
```

> <https://stackoverflow.com/questions/31857365/rails-generate-commands-hang-when-trying-to-create-a-model>

把 `spring` 停掉

    $ spring stop

在重試指令，就好了

```bash
# work
$ rails g -h
```
